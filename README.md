# RealTimeTable таблица позволяет наблюдать изменение позиции курсора и редактировать данные в режиме реального времени разным пользователям.

## Установка

```
npm i
```

## Запуск

**Простой сервер**

```
npm run start:dev
```

**Валидация JS (eslint)**

```
npm run eslint
```

**Валидация CSS**

```
npm run validate-css
```

**Запуск DEV режима (валидаторы + лайв-сервер)**

```
npm run dev-mode
```

**Создание билда**

```
npm run build
```

## Публикация

```
npm run deploy
```
